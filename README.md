目前算力服务器已挂，刚好DeepSeek出来，就顺便又重构一下。该仓库仍在开发中！

最初的目的是想蹭学校的显卡，打造一个类似吐司的stable-diffusion AI生成平台，提供多种AI绘画生成服务，包括但不限于：

- 图像生成
- 模型社区讨论
- 模型训练
- 模型在线部署运行
- 模型在线运行调参

以上需求有部分已经实现，有部分还在开发中。
